Sobre mí

Soy Technical Solutions Specialist en Oracle, con más de una década de experiencia en el ecosistema NetSuite. Inicié mi carrera como profesor de programación en la educación técnica pública (ANEP–CETP), una etapa que me permitió desarrollar una sólida capacidad para comunicar conceptos complejos y comprender cómo las personas interactúan con la tecnología.

A lo largo de mi trayectoria, me desempeñé como Technical Consultant en Techfino LLC (EE. UU.), Principal Consultant y Technical Consultant en NetSuite, y NetSuite Administrator en En La Mano. En estos roles, desarrollé una sólida especialización en SuiteScript 2.x, integraciones REST/SOAP, flujos de trabajo, módulos financieros y operativos, así como en consultoría de implementación para clientes de distintos países.

Cuento con 6 certificaciones de Oracle NetSuite y Celigo, entre ellas SuiteCloud Developer II, SuiteAnalytics User y SuiteFoundation. Actualmente estoy cursando la Especialización en Ciencia de Datos e Inteligencia Artificial en UTEC, en colaboración con el MIT, donde aplico machine learning y análisis de datos a problemas reales dentro de mis proyectos académicos.

Proyectos destacados

Selección de proyectos personales y profesionales

UTEC - Análisis de Siniestros de Tránsito (Montevideo 2022)

Análisis exploratorio de datos abiertos de la Intendencia de Montevideo (2022) sobre siniestros de tránsito. Mediante Python, Pandas, Matplotlib, Seaborn y Plotly identifiqué patrones por día, hora, tipo de vehículo y zona geográfica. Los miércoles y tardes concentran el mayor riesgo; los motociclistas representan el 58% de los siniestros urbanos. Proyecto de la materia Fundamentos de Programación para Ciencia de Datos (UTEC).

Python Pandas Matplotlib Seaborn Plotly
GitHub Colab Dataset

Detección de Campos en Facturas mediante Segmentación Semántica con U-Net

Modelo de segmentación semántica basado en la arquitectura U-Net para detectar y localizar campos clave en imágenes de facturas (fecha, monto, proveedor, etc.). Incluye preprocesamiento de imágenes, entrenamiento del modelo y evaluación con métricas de segmentación. Proyecto de la especialización UTEC.

Python U-Net Deep Learning Computer Vision
GitHub Colab

Posting Job on Upwork — Automatización con IA

En el auge del mercado freelance global, muchos profesionales enfrentan dificultades para establecer tarifas competitivas. Este proyecto desarrolla modelos predictivos para estimar la tarifa horaria promedio (Hourly Rate) en base a habilidades, experiencia y factores del mercado, e identifica qué skills son más valorados por los clientes.

Python Upwork API REST API Automation
GitHub

Broker Communication Software — Middleware de Mensajería

Middleware Express (Node.js / TypeScript) que actúa como punto de integración entre proveedores y sistemas downstream. Recibe payloads de tipo "solicitud", gestiona autenticación, usuarios y datos de perfil, y expone un dashboard operativo y páginas admin. El repositorio alberga dos superficies: un servidor Express legado en JavaScript con endpoints de integración y autenticación, y un servicio de perfiles en TypeScript con API RESTful documentada en Swagger / OpenAPI.

Node.js Express TypeScript REST API Swagger Middleware
GitHub

Fintech CRM Customizations — SuiteCloud (SDF) para NetSuite

Proyecto SuiteCloud (SDF) desarrollado para una fintech uruguaya. Incluye scripts personalizados, objetos y configuraciones de NetSuite para la gestión integral de Leads, integraciones con servicios externos (Clearing, BCU y Web Landing) y mantenimiento de registros. El ciclo de desarrollo sigue las prácticas de SuiteCloud Development Framework: bundles versionados, deployment a entornos sandbox y producción, y gestión de dependencias mediante manifest.xml.

NetSuite SuiteCloud SDF SuiteScript CRM Integrations
